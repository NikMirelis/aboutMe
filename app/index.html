/Инструенты

html

HTML (HyperText Markup Language — «язык гипертекстовой разметки») — это скилет вашего будущего сайта.

CSS

CSS (Cascading Style Sheets - "язык таблиц стилей"). Отвечает за внеший вид: цвета, шрифты, отступы, тени, — это кожа вашего будущего сайта.

JavaScript

JavaScript — это скриптовый язык, который заставляет статическую страницу двигаться без перезагрузки, — мышцы сайта.

Adobe

Photoshop, Illustrator, XD, After Effects, Premier Pro — комплекс программ, необходимых для визуализации.

Sass

Sass — css-препроцессор. Позволяет оптимизировать процесс написания CSS-стилей.

Gulp

Gulp — это таск-менеджер для автоматического выполнения часто используемых задач.

Vue.js

Vue.js — это js-фреймворк, необходим для создания реактивности на странице.

GitHub

GitHub — система контроля версий, позволяет быстро и удобно делиться кодом.
